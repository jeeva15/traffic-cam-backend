### Prerequisites

      Node - Latest
      PostgreSQL

## Installation

To set up this project on your local machine, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/jeeva15/traffic-cam-backend.git
   cd your-project
   ```

2. **To install dependencies:**

   `npm install`


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev


## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
